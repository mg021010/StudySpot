Slide 3. 1단계: 실측 데이터 기반의 상태 판정 기준

// SENSOR THRESHOLDS & DETECTION MATRIX

🔊 1. 음향 분위기 판정 (데시벨 변환식 기반)

데시벨 산출 공식: $dB = 20 \log_{10}(\text{RMS}) + 100$

데시벨 범위 ($dB$) 음향 분위기 판정 카테고리 허용 학습 활동 성격
$dB < 30.0$ 🤫 Deep Focus (정막) 속삭임조차 제한되는 최고 수준의 정숙 구역
$30.0 \le dB < 38.0$ 🌬️ White Noise (백색소음) 일반 자습 및 가벼운 마우스/노트북 타건음 허용
$38.0 \le dB < 45.0$ 🎵 BGM (배경음악) 배경음악 및 가벼운 일상 생활음이 있는 카페 분위기
$dB \ge 45.0$ 🗣️ Active Discussion (토론) 조별 회의, 발표 연습 및 자유 대화 가능 구역

📶 2. 하이브리드 점유 판정 (BLE 디바이스 $N$ & PIR 타이머 $C$)

BLE 탐색 수 ($N$) PIR 타이머 ($C$) 점유 판정 결과 의미 분석 및 오차 보정
$N < 0.5$ $C = 0$ ⚪ Vacant (비어있음) 이용자 부재 및 유휴 공간
$N < 3.5$ $C = 0$ 🟢 Static Low (매우 여유) 소수 이용자가 조용히 개별 학습 중
$N < 3.5$ $C > 0$ 🟢 Dynamic Low (여유) 소수 이용자가 작업 또는 이동 중
$N \ge 3.5$ $C = 0$ 🟡 Static High (정적 밀집) 많은 사람이 있지만 고도의 자습 집중 상태
$N \ge 3.5$ $C > 0$ 🔴 Dynamic High (혼잡) 다수 인원이 활발히 조별 대화/이동 중
🎙️ Slide 3 발표 대본
"수집된 센서 수치들은 다음과 같은 논리 기준을 가집니다. 마이크 전압값은 로그 변환을 거쳐 30데시벨 미만의 초정막 상태부터 45데시벨 이상의 토론 상태까지 분류됩니다. 특히 점유 상태 판정에 공을 들였습니다. 일반 PIR 센서는 사용자가 꼼짝 않고 공부하면 오인식되어 빈방 처리를 해버립니다. 저희는 주변의 블루투스 신호 대수(N)와 10초 쿨다운을 결합한 알고리즘을 얹어, 블루투스 기기가 3.5대 이상 탐색되는데 정지 상태라면 이를 'Static High', 즉 '조용하게 초집중 중인 정적 밀집 상태'로 오차 없이 감지해 냅니다."

Slide 4. 2단계: 종합 적합도 매칭 공식

// ALGORITHM MODEL VARIABLES & MATRIX

최종 매칭률(%)은 기본 점수 50점에 소음 가중치($W_{acoustic}$)와 혼잡도 가중치($W_{occupancy}$)를 합산하고, 쾌적 온도를 이탈하면 감점 페널티($P_{temp}$) 5점을 차감하여 구합니다.

$$\mathbf{SCORE = S_{base} + W_{acoustic} + W_{occupancy} - P_{temp}}$$
S_base (50점)
기본 기준 점수
모든 연산의 버팀목이 되는 기본 상수입니다. 가감 수식의 밸런스를 잡고, 스코어 범위 안정화를 도모합니다.
W_acoustic (-40 ~ +40)
음향 분위기 가중치
목적별 최적 소음 단계에는 가중치 보너스를 부여하고, 맞지 않는 환경(예: 자습 시 대화 소음)에는 페널티 감점을 부여합니다.
W_occupancy (-30 ~ +18)
하이브리드 점유 가중치
정적/동적 밀집 상태를 고려합니다. 조용한 자습 환경에는 여유로운 상태 가중치를 주어 상단 큐레이션을 돕습니다.

🌡️ 학습 집중을 돕는 온도 페널티 ($P_{temp}$) 적용

인간의 학습 능률 최적 온도 범위인 $21.0^\circ\text{C} \sim 24.5^\circ\text{C}$를 이탈하면 감점을 줍니다.

  • $P_{temp} = 0\text{점}$ 감점 없음: 온도 $T$가 쾌적 범위 ($21.0^\circ\text{C} \le T \le 24.5^\circ\text{C}$) 안인 경우
  • $P_{temp} = 5\text{점}$ 감점 적용: 온도 $T$가 범위를 이탈 ($T < 21.0^\circ\text{C}$ 또는 $T > 24.5^\circ\text{C}$)한 경우
🎙️ Slide 4 발표 대본
"저희 적합도 스코어 산출 모델은 기본 점수 50점에 소음 가중치, 혼잡도 가중치를 곱하는 게 아닌 '합산(Sum)'하고, 실내 온도가 최적 범위를 벗어나면 5점을 깎아버리는 페널티 차감 연산 구조를 가집니다. 공부 능률에 최적인 21도와 24.5도 사이의 쾌적한 공간을 사용자에게 추천하기 위해 온도 차감 변수를 두었고, 최종 스코어는 화면 표시 상의 신뢰도를 위해 최소 10%에서 최대 98% 범위 내로 고정하도록 바인딩 필터를 마지막 단계에 걸어 두었습니다."
